
## Available Scripts

Before you can start this app You must intall dependencies first.
In the project src folder you can run:

### `npm install`

In the project directory, you can run:
### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


This app purpose is to use urls which contains JSON data and displays post Title in frontpage.
When you click post title you get post body and username of the post maker.

Clicking username you can find most important information about user.
